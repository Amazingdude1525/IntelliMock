# 🌌 IntelliMock: Elite AI Interview Platform

![IntelliMock Banner](https://images.unsplash.com/photo-1614850523296-d8c1af93d400?auto=format&fit=crop&q=80&w=2070)

> **Elevate your career readiness with real-time biometric feedback and industry-standard AI intelligence.**

IntelliMock is a high-fidelity recruitment preparation platform that simulates real-world interview environments. Featuring **Elite Mode**, it provides an immersive 3D-integrated experience with precise behavioral analytics.

## 🚀 Key Features

- **🎭 Elite 3D Environment**: Immersive Spline-integrated interface with a 3D Interviewer Panel.
- **🎙️ Neural Voice Mode**: Realistic, low-latency voice interactions powered by Groq Llama 3.3.
- **📏 Biometric Precision**: Real-time tracking of eye contact, posture, focus, and calmness using industry-grade neural networks.
- **📄 Resume-Adaptive AI**: Context-aware questioning based on your specific career history and uploaded documents.
- **📊 The Perfect Report**: Comprehensive post-interview analytics including performance radar charts and 3-stage career roadmaps.
- **👤 Personalized Identity**: Geolocation-aware greetings and customized onboarding flows.

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite, TypeScript, Tailwind CSS v4, Framer Motion.
- **3D Graphics**: Spline (@splinetool/react-spline).
- **Backend Infrastructure**: Node.js, Express, Drizzle ORM.
- **Database**: Neon (PostgreSQL) with Serverless Scaling.
- **Intelligence**: Groq AI (Llama 3.3 70B) for neural-link streaming.
- **Authentication**: Clerk (Google / GitHub Social Sync).
- **Biometrics**: Face-API.js for neural-network-based expression analysis.

## 📦 Getting Started

### Prerequisites
- Node.js 18+
- NPM or PNPM

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Amazingdude1525/IntelliMock.git
   cd IntelliMock
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Copy `.env.example` to `.env` and provide your keys:
   - `VITE_CLERK_PUBLISHABLE_KEY`
   - `VITE_API_URL`

4. Launch the development server:
   ```bash
   npm run dev
   ```
